# lsdeno
lsdeno is the ls command which reimplemented by deno

## requirements
- [deno v1.17.1](https://deno.land/)

## build
```
$ make build
```

## usage
```
$ ./build/lsdeno
Makefile        build           lsdeno.ts       misc            src             test
```

## install
```
$ make install
```

## uninstall
```
$ rm /usr/local/bin/lsdeno
```
